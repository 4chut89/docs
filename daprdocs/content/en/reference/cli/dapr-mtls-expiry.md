## Description

Checks the expiry of the root certificate

## Usage
```bash
dapr mtls expiry [flags]
```

## Examples

### Check expiry of Kubernetes certs
```bash
dapr mtls expiry
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--help`, `-h` | | | help for expiry |
